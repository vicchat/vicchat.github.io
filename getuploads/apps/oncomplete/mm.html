Information to setup:

When you download the file, you will see it comes in a zip format.

Step 1: Click on it and once it is done downloading you will see it has a file in the file explorer. Drag this on to your desktop.

Step 2: Go to C:\Users\Your Name in your file explorer, and drag the folder into this directory from your desktop.

Step 3: To continue, install Node.js.

Step 4: Once Node.js is installed, open Node.js command prompt.

Step 5: Type into the command prompt "cd MayanMan", then type "npm start". The app should open, and while it is open, DO NOT close Node.js command prompt.
