DotLauncher

The best way to add mods to The Random Puffy Thingy and the Big Scary Void. Installs *.puffypkg files.

Compatibility

Electron Edition: Yes
Chunk Desktop: No
Chunk Mobile: No

Type: Modified Version

Modified Version Information

Version Modified: 1.3_s5
Capabilities: Install puffypkg mods, Additional modding capabilities.

About

DotLauncher is yet another way to install mods to The Random Puffy Thingy and the Big Scary Void. Unlike other modding options, DotLauncher has more creator features, making it easier to add more powerful mods to the game.

It is very simple to install, just open and extract the downloaded zip file. Put the installer file (DotLauncher.exe) somewhere you can access it, you will need it to install mods. Move the DotLauncher folder to C:\Users\puffyConfig\mods and mention it in your options.js file. Afterwards, start DotLauncher.exe, and enter the path of the mod.puffypkg file in the box (which is most likely C:\Users\USERNAME\Downloads\MODNAME).

To play a mod, launch Puffy Launcher version “Mod Launcher” and double-click DotLauncher. After this, you can type the mod's name in the box and click “Play”. No need to specify the path here, you have already installed it. After you have installed the Modified Version, there will be no need to change a JavaScript file; DotLauncher doesn't require one to run mods. Please keep in mind you will still need the options.js file in the puffyConfig directory.

Future updates will include update notifications and automatically installing the modified version to the launcher. For now, check back to this page constantly once you update The Random Puffy Thingy and the Big Scary Void Launcher to version 1.0.9, and an update should arrive within a few days. After that, you will be notified when you start DotLauncher if there is a update available.

Multirealms

Multirealms adds more dimensions to your platforms.

Compatibility

Electron Edition: Yes
Chunk Desktop: No
Chunk Mobile: No

Type: DotLauncher Mod

About

Go to the top-right side of the platform of which you spawn on, and you should see four portals. The yellow and red portal will take you to a “Nether” dimension, a dimension of which mimics the colour and outline of it's portal (because all the dimensions do). The purple and green portal takes you to an “End” dimension. The yellow and blue portal goes to a “Sky” dimension, and the black and purple portal takes you to a “Lair” dimension. To return to the “Overworld” dimension, go through the grey and green portal located in each dimension (you will come through it when you enter a dimension). Just move the puffy thingy to the center of the portal, and you should be teleported inside.
